📘 Panduan Lengkap ARIKA

Panduan Penggunaan ARIKA

Panduan ini menjelaskan alur penggunaan ARIKA setelah pengembangan terbaru, meliputi pengisian jurnal, lembur, dashboard pegawai, alarm pengingat, catatan atasan, verifikasi ketua tim, dashboard jurnal, dashboard lembur, rekap jurnal pegawai, SPK lembur, pengumuman, agenda, survei, dan pengelolaan role.

Pegawai

Isi jurnal, pantau catatan, lembur pribadi, reminder, dan dashboard saya.

Ketua Tim

Review jurnal, verifikasi lembur, dan melihat rekap terbatas.

Admin Utama

Kelola master data, monitoring, pengumuman, agenda, survei, dan dokumen.

Reminder

Alarm suara perlu diaktifkan satu kali setelah login.

Mode Fokus Panduan
Bagian lain dibuat abu-abu agar Anda fokus pada topik yang dipilih.

👤 Panduan untuk Personil Pegawai

Bagian ini digunakan oleh pegawai non-admin untuk mencatat kegiatan, memantau riwayat, mengelola lembur, dan menindaklanjuti catatan atasan.

Akses Pegawai

1. Login Pegawai

Pilih nama pada daftar pegawai, masukkan NIP, lalu klik Masuk Portal. Pastikan nama dan NIP sesuai dengan data pada Sheet Pegawai.

2. Beranda Pegawai

Beranda menampilkan informasi ringkas, agenda, pengumuman, rencana pribadi, reminder pengisian jurnal, alert catatan atasan, dan tombol bantuan WhatsApp.

3. Isi Jurnal Harian

Buka tab Isi Jurnal, isi tanggal, kategori, uraian kegiatan, status kegiatan, dan link data dukung bila ada. Setelah disimpan, data masuk ke Sheet Jurnal dan dapat dilihat di riwayat.

4. Jurnal Lembur

Untuk kegiatan lembur, centang Kegiatan Lembur. Isi jam mulai, jam selesai, nomor surat tugas, dan data dukung. Dashboard lembur hanya membaca jurnal yang memiliki tanda lembur eksplisit.

5. Riwayat Jurnal

Tab Riwayat Jurnal berada di sebelah kanan tab Isi Jurnal. Pegawai dapat memeriksa, mencari, menyunting, atau menghapus jurnal miliknya sesuai kebutuhan koreksi.

6. Laporan Lembur

Tab ini menampilkan data lembur milik pegawai login. Gunakan untuk memantau riwayat lembur pribadi dan memastikan kelengkapan surat tugas atau data dukung.

7. Dashboard Saya

Dashboard Saya menampilkan ringkasan jurnal pribadi, lembur pribadi, total jam lembur, tren aktivitas, catatan atasan, dan monev unit ringkas tanpa membuka data sensitif admin.

8. Catatan Atasan

Jika ketua tim/admin memberi catatan pada jurnal, pegawai akan melihat alert di Beranda hanya selama catatan tersebut belum ditindaklanjuti. Setelah pegawai menekan tombol Tandai Sudah Ditindaklanjuti, alert catatan tidak akan muncul lagi untuk jurnal tersebut.

9. Rencana Saya

Gunakan Rencana Saya untuk mencatat rencana kegiatan beserta jam reminder. Rencana akan memicu alert, alarm suara, dan notifikasi saat tanggal dan jam reminder sudah tiba, selama fitur suara/notifikasi sudah diaktifkan.

10. Survei ARIKA

Isi survei bulanan untuk memberi masukan terkait kemudahan, kemanfaatan, efisiensi, kepuasan, kendala, dan saran pengembangan aplikasi.

Catatan jam lembur: Jam lembur pada Dashboard Saya dihitung dari durasi jam lembur pada absensi dan mengikuti pembulatan yang berlaku pada sistem ARIKA.

🔔 Panduan Alarm dan Reminder

ARIKA dapat memberi pengingat visual, suara, dan Pusat Reminder untuk membantu pegawai tidak melewatkan jurnal, catatan atasan, rencana pribadi, pengumuman, dan agenda.

Aktifkan Suara

Setelah login, klik Aktifkan Suara satu kali. Browser tidak mengizinkan audio otomatis sebelum pengguna melakukan interaksi.

Tes Suara

Gunakan tombol Tes untuk memastikan alarm aktif. Jika suara Bahasa Indonesia tidak tersedia, ARIKA memakai nada chime ARIKA.

Pusat Reminder

Gunakan tombol Buka Pusat Reminder di Beranda atau tombol Pusat Reminder melayang untuk melihat semua reminder aktif tanpa perlu izin notifikasi browser.

Tab ARIKA Tetap Terbuka

Suara dan Pusat Reminder bekerja selama tab ARIKA masih terbuka, browser tidak menidurkan tab, dan audio tidak diblokir.

Agar Alarm dan Reminder Lebih Stabil

  • ✅ Login ARIKA terlebih dahulu.
  • ✅ Klik Aktifkan Suara satu kali setelah login.
  • ✅ Jangan menutup tab ARIKA.
  • ✅ Jangan melakukan mute pada tab ARIKA.
  • ✅ Pastikan volume laptop/PC aktif dan tidak terlalu kecil.
  • ✅ Pada Chrome/Edge, nonaktifkan atau beri pengecualian Memory Saver untuk halaman ARIKA jika alarm sering tidak berbunyi.

Catatan: alarm dan Pusat Reminder ARIKA bekerja dari halaman web yang sedang terbuka. Jika tab ARIKA ditutup, browser ditutup, komputer sleep, atau tab ditidurkan oleh browser, pengingat dapat terlambat atau tidak berbunyi.

Pusat Reminder sebagai Pengganti Notifikasi Browser

Akses Cepat

Tombol Buka Pusat Reminder pada Beranda dibuat membuka panel di bagian atas agar terlihat jelas saat ARIKA disematkan ke Google Sites.

Status Reminder

Badge angka akan muncul jika ada reminder yang belum dibaca. Reminder dapat ditandai dibaca agar tampilan tetap rapi.

Suara Alarm

Aktifkan suara satu kali setelah login. Jika belum aktif, tombol Aktifkan Suara akan berkedip agar mudah terlihat.

Catatan: fitur notifikasi pop-up browser dinonaktifkan agar ARIKA tidak bergantung pada izin browser. Fokus pengingat dipindahkan ke Pusat Reminder, banner visual Beranda, dan suara alarm.

✅ Panduan Ketua Tim / Verifikator

Ketua tim dapat melakukan review jurnal dan verifikasi lembur tanpa menggunakan akun admin utama.

Role Ketua Tim

1. Aktivasi Role

Admin mengisi kolom Peran pada Sheet Pegawai dengan nilai Ketua Tim untuk Rosana Anna Ashari dan Imam Taufik. Untuk pembatasan unit, isi kolom Cakupan Unit bila diperlukan. Jika kolom tersebut kosong, ARIKA memakai aturan khusus: Rosana Anna Ashari memantau Kimia Pangan & Mikrobiologi, sedangkan Imam Taufik memantau unit selain dua fungsi tersebut.

2. Menu Ketua Tim

Setelah login, menu Ketua Tim akan muncul otomatis untuk akun dengan Peran Ketua Tim. Menu ini berbeda dari Admin Utama dan hanya menampilkan fitur yang diperlukan untuk pemeriksaan serta pengelolaan pengumuman/agenda.

3. Dashboard Jurnal

Digunakan untuk melihat tren kegiatan, status jurnal, catatan kegiatan, dan ringkasan monitoring. Nama tab lama Dashboard Monev sudah diganti menjadi Dashboard Jurnal.

4. Dashboard Lembur

Digunakan untuk memeriksa jurnal bertanda lembur, melihat KPI lembur, rekap personil, kelengkapan data, dan daftar verifikasi laporan lembur.

5. Rekap Jurnal Pegawai

Ketua tim dapat membuka tab Rekap Jurnal Pegawai untuk melihat rekap jurnal. Tombol unduh semua data tetap dibatasi untuk Admin Utama.

6. Mengisi Catatan

Klik tombol Review pada jurnal atau lembur, pilih status evaluasi, tulis catatan, lalu simpan. Pegawai akan mendapat alert catatan di Beranda dan Dashboard Saya.

Batas akses Ketua Tim: Ketua Tim tidak dapat mengakses Manajemen Pegawai, Survei, pengaturan admin utama, dan tombol tertentu yang dikhususkan untuk Admin Utama. Ketua Tim tetap dapat membuat Papan Pengumuman dan Agenda Kegiatan sesuai kewenangan yang diberikan.

🛠️ Panduan untuk Administrator Utama

Admin Utama memiliki akses penuh terhadap dashboard, master data, pengumuman, agenda, survei, rekap, SPK, dan pengelolaan role.

Akses Penuh

Dashboard Jurnal

Pantau total jurnal, ketuntasan, tren aktivitas, distribusi status, catatan kegiatan, laporan bulanan, top kontributor, dan rekap monev jurnal.

Dashboard Lembur

Pantau entri lembur, total jam, personil lembur, kelengkapan berkas, grafik lembur, rekap personil, dan daftar verifikasi dengan paginasi 10 laporan per halaman.

SPK Lembur

SPK Harian dan SPK Rentang/Kolektif berada pada Dashboard Lembur. Format SPK tidak memakai header/footer gambar dan tidak memakai kop teks lama. Nomor SPK memakai kode KU.01.03.58B.

Rekap Jurnal Pegawai

Gunakan tab ini untuk melihat seluruh jurnal pegawai, mencari data, memeriksa riwayat, dan mengunduh data sesuai kebutuhan monitoring admin.

Manajemen Pegawai

Tambah atau perbarui data pegawai. Pastikan kolom Nama, NIP, Status, Fungsi Kerja, Pangkat, dan Peran diisi benar. Peran dapat diisi Pegawai, Ketua Tim, atau Verifikator sesuai kewenangan.

Papan Pengumuman

Buat pengumuman dengan judul, jenis, tanggal mulai, tanggal selesai, waktu mulai, target fungsi, dan isi. Pengumuman aktif akan muncul pada Beranda Pegawai.

Agenda Kegiatan

Input agenda kegiatan, lokasi/media, waktu mulai, keterangan, dan peserta. Agenda dapat ditampilkan pada Beranda dan kalender kegiatan pegawai.

Survei dan Tindak Lanjut

Pantau hasil survei kepuasan, kemanfaatan, efisiensi, kendala, dan saran. Data dapat digunakan sebagai bahan monitoring dan evaluasi pengembangan ARIKA.

❓ FAQ ARIKA

Kumpulan pertanyaan yang paling sering muncul setelah pengembangan ARIKA, mulai dari login, jurnal, lembur, dashboard, role verifikator, alarm, hingga dokumen SPK.

Pertanyaan Umum

A. Login, Akun, dan Role

Saya login sebagai pegawai, kenapa menu admin tidak muncul?⌄

Menu admin hanya muncul untuk Admin Utama. Pegawai biasa hanya melihat menu pegawai seperti Beranda, Isi Jurnal, Riwayat Jurnal, Laporan Lembur, Dashboard Saya, dan Panduan.

Ketua tim sudah diberi peran, tetapi menu Verifikator belum muncul. Apa yang harus dilakukan?⌄

Pastikan kolom Peran pada Sheet Pegawai berisi Ketua Tim untuk Rosana/Imam atau Ketua Tim untuk pemeriksa lain, nama dan NIP sama dengan akun yang digunakan login, lalu logout dan login ulang agar data role terbaca kembali.

Apa perbedaan Pegawai, Ketua Tim, dan Admin Utama?⌄

Pegawai mengelola jurnal dan dashboard pribadinya. Ketua Tim memiliki tambahan menu Ketua Tim untuk memantau jurnal unit binaan, review jurnal, verifikasi lembur, serta membuat pengumuman dan agenda. Verifikator tetap tersedia sebagai role opsional untuk pemeriksa lain. Admin Utama memiliki akses penuh ke seluruh dashboard, master pegawai, pengumuman, agenda, survei, rekap, dan dokumen.

Apakah Ketua Tim perlu menggunakan akun admin utama?⌄

Tidak. Ketua Tim cukup login sebagai pegawai biasa menggunakan nama dan NIP masing-masing. Jika kolom Peran sudah diisi Ketua Tim, menu Ketua Tim akan muncul otomatis.

B. Jurnal Harian dan Riwayat

Bagaimana cara mengisi jurnal harian?⌄

Buka tab Isi Jurnal, pilih tanggal, kategori, status pekerjaan, uraian kegiatan, lalu simpan. Jika kegiatan merupakan lembur, aktifkan tanda/checkbox lembur dan lengkapi jam serta nomor surat tugas jika diperlukan.

Kenapa jurnal saya tidak muncul di Riwayat Jurnal?⌄

Klik tombol refresh/sinkron data, pastikan koneksi stabil, lalu cek filter bulan/tanggal. Jika masih kosong, pastikan jurnal sudah tersimpan di Google Sheet dengan nama/NIP yang sama dengan akun login.

Apakah jurnal yang sudah tersimpan bisa disunting?⌄

Bisa, selama tombol edit tersedia pada riwayat. Penyuntingan memakai ID Jurnal agar data lama diperbarui, bukan membuat duplikasi baru.

Apa yang harus dilakukan jika ada catatan atasan pada jurnal?⌄

Buka alert pada Beranda atau tab Dashboard Saya, baca catatan atasan, lakukan perbaikan/tindak lanjut sesuai arahan, lalu klik Tandai Sudah Ditindaklanjuti. Setelah status tindak lanjut tersimpan, reminder catatan tidak akan muncul lagi untuk jurnal tersebut.

C. Lembur dan SPK

Data seperti apa yang masuk Dashboard Lembur?⌄

Hanya jurnal yang memiliki tanda lembur eksplisit, misalnya kolom Lembur? bernilai TRUE, Ya, Iya, 1, atau Lembur. Sistem tidak lagi menebak dari uraian, jam, atau surat tugas saja.

Kenapa data lembur saya tidak masuk dashboard?⌄

Periksa apakah saat mengisi jurnal tanda lembur benar-benar aktif. Jika data lama hanya memiliki jam lembur atau surat tugas tetapi kolom Lembur? kosong, data tersebut tidak akan dihitung sebagai lembur.

Bagaimana jam lembur dihitung?⌄

Jam lembur pada Dashboard Saya mengikuti durasi jam lembur pada absensi. Pada verifikasi SPK, durasi yang tampil mengikuti data jam mulai dan jam selesai yang tersimpan pada jurnal lembur.

Di mana tombol SPK Harian dan SPK Kolektif?⌄

Tombol SPK Harian dan SPK Kolektif/Rentang berada pada Dashboard Lembur. Dokumen SPK mengambil data jurnal yang bertanda lembur sesuai filter tanggal dan fungsi/lab yang dipilih.

Kenapa SPK tidak memakai header/footer gambar?⌄

Sesuai revisi terbaru, SPK Harian dan SPK Kolektif dibuat tanpa header/footer gambar dan tanpa kop teks lama. Isi dokumen dimulai dari judul Surat Perintah Kerja Lembur.

D. Dashboard Pegawai, Jurnal, dan Lembur

Apa fungsi Dashboard Saya?⌄

Dashboard Saya menampilkan ringkasan pribadi pegawai: jumlah jurnal, status pekerjaan, lembur pribadi, catatan atasan, tren aktivitas, dan monev unit ringkas tanpa membuka data sensitif admin.

Apa bedanya Dashboard Jurnal dan Rekap Jurnal Pegawai?⌄

Dashboard Jurnal berisi ringkasan dan grafik monitoring. Rekap Jurnal Pegawai berisi daftar data jurnal yang lebih rinci untuk ditelusuri, difilter, dan ditinjau.

Kenapa grafik terlihat kosong?⌄

Biasanya karena filter bulan/tanggal tidak sesuai dengan data yang tersedia. Coba pilih Semua Bulan, klik Sinkron/Refresh Data, dan pastikan data jurnal sudah tersimpan di Sheet.

Apakah Ketua Tim bisa melihat semua data pegawai?⌄

Ketua Tim dapat melihat Dashboard Jurnal, Dashboard Lembur, dan Rekap Jurnal Pegawai sesuai unit binaannya. Rosana Anna Ashari dibatasi pada Kimia Pangan & Mikrobiologi, sedangkan Imam Taufik dibatasi pada unit selain Kimia Pangan & Mikrobiologi. Ketua Tim juga dapat membuat Papan Pengumuman dan Agenda Kegiatan. Fitur manajemen dan ekspor tertentu tetap dibatasi untuk Admin Utama.

E. Alarm, Reminder, Pengumuman, dan Agenda

Kenapa alarm tidak berbunyi setelah membuat reminder hari ini?⌄

Klik Aktifkan Suara setelah login, pastikan volume laptop/PC aktif, tab ARIKA masih terbuka, tab ARIKA tidak di-mute, dan browser tidak memblokir audio. Setelah reminder dibuat, sistem akan melakukan pengecekan ulang; gunakan tombol Tes Suara dan Pusat Reminder untuk memastikan pengingat aktif. Jika Chrome/Edge memakai Memory Saver dan alarm sering tidak berbunyi, berikan pengecualian untuk halaman ARIKA agar tab tidak ditidurkan otomatis.

Apakah alarm tetap berbunyi saat saya membuka tab lain?⌄

Bisa, selama tab ARIKA masih terbuka dalam browser yang sama dan suara sudah diaktifkan. Untuk menghindari kendala izin browser, pengingat utama diperkuat melalui Pusat Reminder, banner visual Beranda, dan suara alarm. Jika tab ARIKA ditutup atau komputer masuk mode sleep, alarm dan reminder tidak akan berjalan.

Bagaimana cara membuka Pusat Reminder?⌄

Klik tombol Buka Pusat Reminder pada Beranda atau tombol Pusat Reminder yang melayang di kiri bawah. Di dalamnya tersedia ringkasan reminder aktif, tombol Aktifkan Suara, Tes Suara, Tandai Dibaca, dan tombol menuju Beranda. Badge angka akan muncul jika ada reminder yang belum dibaca.

Seberapa sering alarm berbunyi?⌄

Sistem mengecek reminder secara berkala. Reminder pribadi akan mengikuti tanggal dan jam reminder yang dipilih. Reminder pribadi, jurnal, dan catatan tidak dibunyikan terus-menerus; ada jeda agar tidak mengganggu. Agenda yang mendekati waktu mulai dapat diingatkan lebih sering.

Kenapa suara manusia tidak tersedia atau berubah menjadi beep?⌄

ARIKA hanya memakai suara Bahasa Indonesia jika tersedia di browser/perangkat. Jika tidak tersedia, ARIKA memakai nada chime ARIKA sebagai pengingat cadangan.

Apa saja yang bisa memicu reminder di Beranda?⌄

Reminder dapat muncul untuk jurnal hari ini yang belum diisi, jurnal terlewat hari kerja, catatan atasan, rencana pribadi, pengumuman aktif, dan agenda kegiatan yang mendekati waktu mulai.

F. Admin, Data, dan Teknis

Apa yang harus dilakukan setelah kode HTML diganti di Google Sites?⌄

Tempel kode HTML terbaru pada embed Google Sites, klik simpan, lalu publish. Setelah itu buka ulang halaman ARIKA dan lakukan refresh browser agar cache lama tidak terbaca.

Kapan Apps Script perlu diganti atau deploy ulang?⌄

Apps Script perlu diperbarui jika ada perubahan backend, misalnya penambahan kolom Peran, perubahan API, atau penyimpanan data baru. Jika hanya tampilan HTML yang berubah, biasanya cukup mengganti kode embed.

Apakah data ARIKA aman jika ada banyak pengguna?⌄

ARIKA membatasi tampilan berdasarkan role di frontend. Untuk kedisiplinan operasional, admin perlu menjaga akses Google Sheet dan Apps Script agar hanya petugas yang berwenang yang bisa mengubah data sumber.

Kenapa data yang baru diinput belum langsung muncul?⌄

Data mungkin masih menggunakan cache sementara. Klik Sinkron/Refresh Data, buka ulang tab, atau logout dan login kembali jika data masih belum berubah.

Bagaimana cara melaporkan kendala ke admin ARIKA?⌄

Gunakan tombol Bantuan ARIKA di kanan bawah layar. Pesan bantuan akan membawa konteks halaman yang sedang dibuka, sehingga admin lebih mudah menelusuri masalah.

💬 Bantuan dan Kendala Umum

Data tidak muncul

Klik tombol sinkron/refresh data, pastikan koneksi internet stabil, dan pastikan data pada Google Sheet sudah tersimpan dengan header kolom yang benar.

Alarm tidak berbunyi

Klik Aktifkan Suara dan Tes Suara. Pastikan tab ARIKA masih terbuka, tab tidak di-mute, volume laptop/PC aktif, dan browser tidak memblokir audio. Jika sering tidak berbunyi, matikan atau beri pengecualian Memory Saver Chrome/Edge untuk halaman ARIKA.

Menu Verifikator tidak muncul

Pastikan kolom Peran pada Sheet Pegawai berisi Ketua Tim untuk Rosana/Imam, atau Verifikator untuk pemeriksa lain, lalu logout dan login kembali.

Lembur tidak masuk dashboard

Pastikan jurnal tersebut memiliki tanda lembur, misalnya kolom Lembur? bernilai TRUE/Ya/Lembur. Dashboard tidak menebak dari uraian, jam, atau surat tugas saja.

Masih butuh bantuan?

Gunakan tombol Bantuan ARIKA di kanan bawah layar. Pesan WhatsApp akan otomatis berisi identitas pengguna dan halaman terakhir yang sedang dibuka.
